# Melody Lin - Portfolio Website

A modern, responsive portfolio website showcasing data analytics projects and research work.

## Features

- **Profile Section**: Professional introduction with links to LinkedIn and GitHub
- **Data Projects**: Filterable project showcase (Python/SQL categories)
- **Research Projects**: Academic and industry research publications
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, minimalist design with smooth animations

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- No external dependencies

## Structure

```
portfolio-website/
├── index.html          # Main HTML file
├── style.css           # Styling
├── script.js           # Tab navigation and filtering
└── README.md           # This file
```

## Local Development

Simply open `index.html` in your web browser. No build process or server required.

## Deployment

This site can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
